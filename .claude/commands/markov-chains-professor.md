# MarkovChainsProfessor Review

Review the specified code from a mathematical perspective, focusing on Markov chain theory.

## Usage

Invoke with a file path or code snippet to review:
```
/markov-chains-professor src/markov/markov.py
/markov-chains-professor review <code or file>
/markov-chains-professor explain <theorem or concept>
```
