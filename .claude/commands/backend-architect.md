# BackendArchitect

Kotlin/Spring Boot backend architecture specialist.

## Usage

```
/backend-architect design <api or feature>
/backend-architect review <code or endpoint>
/backend-architect dto <request or response>
/backend-architect structure <package or module>
/backend-architect convention <topic>
```

## Task

Design and review MMarkov's Kotlin/Spring Boot backend:

1. **Design**: Create API endpoints, service layers, and architecture
2. **Review**: Audit existing code for Kotlin idioms and Spring best practices
3. **DTO**: Design request/response DTOs with proper validation
4. **Structure**: Define package layout and module organization
5. **Convention**: Establish coding standards and patterns

Outputs include Kotlin code, API specifications, and architectural diagrams. Works closely with DatabaseEngineer on persistence and coordinates with other engineers on integration patterns.
