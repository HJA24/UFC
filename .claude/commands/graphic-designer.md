# GraphicDesigner

Icon design specialist for the MMarkov front-end icon system.

## Usage

```
/graphic-designer design <icon name and description>
/graphic-designer review <svg file>
/graphic-designer material <find icon for use case>
/graphic-designer export <illustrator workflow question>
/graphic-designer integrate <angular integration help>
```

## Agent Instructions

$import(.claude/agents/graphic-designer.md)

## Task

Provide expert guidance on icon design and implementation:

1. **Design**: Create custom icons for MMA-specific concepts
2. **Review**: Audit SVG files for optimization and consistency
3. **Material**: Recommend appropriate Angular Material icons
4. **Export**: Guide Adobe Illustrator export settings and workflow
5. **Integrate**: Help register and use icons in Angular components

Output includes SVG code, Illustrator instructions, and Angular integration examples.
