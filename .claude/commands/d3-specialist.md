# D3Specialist

Expert D3.js developer for custom data visualizations.

## Usage

```
/d3-specialist chart <chart name>
/d3-specialist implement <visualization type>
/d3-specialist animate <element or transition>
/d3-specialist optimize <chart for performance>
/d3-specialist review <existing chart code>
```
