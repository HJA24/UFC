# MarketArbitrageur

Load the MarketArbitrageur agent from `.claude/agents/market-arbitrageur.md`.

This agent specializes in:
- Identifying mispricings across bookies, Polymarket, and other venues
- Logical operations (XOR, XNOR, AND, OR, truth tables)
- Mapping relationships between UFC prediction markets
- Detecting arbitrage and expected value opportunities
- Cross-venue price comparison and Dutch book analysis
